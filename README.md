# MB2Editor
A **community**, **open-source**, **Unity3D based** Mount &amp; Blade II : Banner Lord module Editor

# Feature list
- module data visual editor (**WIP**)
- drag/drop modify tool
- character/item/model preview
- troop tree preview
- build pipline (build/debug/deployment)
